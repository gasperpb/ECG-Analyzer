   ├── Exibição de estatísticas

COMO EXECUTAR?
==============

1. Instale Maven (se não tiver)
   
2. No terminal, navegue até o projeto:
   cd "c:\Users\Achilles\Documents\Projetos\java\firstprojet\firstprojet"

3. Compile e execute:
   mvn spring-boot:run

4. Abra no navegador:
   http://localhost:8080

5. Pronto! Analise seus dados de ECG! 🎉

RECURSOS DA PÁGINA WEB
======================

📤 Upload
  └─ Carregue arquivo CSV com dados ECG
  └─ Análise automática
  └─ Resultados em tempo real

🎲 Simulação
  └─ Gere dados fictícios
  └─ Teste sem arquivo
  └─ Duração customizável

📊 Gráfico Interativo
  └─ Visualize o sinal ECG (azul)
  └─ Picos detectados em vermelho
  └─ Zoom e pan
  └─ Lendas e tooltips

📈 Estatísticas
  └─ BPM (Batidas por minuto)
  └─ Número de batidas
  └─ Voltagens min/máx/média
  └─ Status de saúde

🚨 Detecção de Anomalias
  └─ Bradicardia (BPM < 60)
  └─ Taquicardia (BPM > 100)
  └─ Arritmias (variação anormal)
  └─ Alertas em tempo real

API REST DISPONÍVEL
===================

POST /api/ecg/analyze
  └─ Upload e análise de arquivo
  └─ Retorna: BPM, picos, status, gráfico

GET /api/ecg/simulate?duration=10
  └─ Gera dados simulados
  └─ Duração: 5-60 segundos

GET /api/ecg/health
  └─ Verifica status do servidor
