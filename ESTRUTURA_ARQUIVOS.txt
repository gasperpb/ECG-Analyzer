ARQUIVOS PRINCIPAIS (Ordem de Importância)
==========================================

1️⃣  GUIA_RAPIDO.md
    └─ Leia isto primeiro!
    └─ Instruções passo a passo

2️⃣  pom.xml
    └─ Dependências do projeto
    └─ Spring Boot, Gson, etc

3️⃣  src/main/resources/templates/index.html
    └─ Interface web (frontend)
    └─ Onde você vê os gráficos

4️⃣  src/main/java/com/ecg/ECGApplication.java
    └─ Aplicação principal
    └─ Inicia o servidor Spring

5️⃣  src/main/java/com/ecg/controller/ECGController.java
    └─ API REST
    └─ /api/ecg/analyze
    └─ /api/ecg/simulate

COMO USAR OS ARQUIVOS
=====================

PASSO 1: Leia o guia
    └─ Abra: GUIA_RAPIDO.md

PASSO 2: Instale Maven
    └─ Execute: instalar-maven.bat (Windows)
    └─ Ou: instalar-maven.sh (Linux/Mac)

PASSO 3: Compile o projeto
    └─ Execute: mvn clean install

PASSO 4: Inicie o servidor
    └─ Execute: mvn spring-boot:run

PASSO 5: Abra no navegador
    └─ Acesse: http://localhost:8080
