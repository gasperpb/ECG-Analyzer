🚀 PARA COLOCAR EM UMA PÁGINA WEB - 3 PASSOS PRINCIPAIS
========================================================

PASSO 1: INSTALAR MAVEN (gerenciador de dependências)
────────────────────────────────────────────────────

Windows:
  1. Baixe: https://maven.apache.org/download.cgi
  2. Descompacte em: C:\Program Files\
  3. Adicione ao PATH do Windows
  4. Reinicie o terminal

Linux/Mac:
  sudo apt install maven    (Ubuntu)
  brew install maven        (Mac)

Verificar:
  mvn -v


PASSO 2: COMPILAR E PREPARAR A APLICAÇÃO
──────────────────────────────────────────

No terminal, execute:

  cd "c:\Users\Achilles\Documents\Projetos\java\firstprojet\firstprojet"
  mvn clean install -DskipTests

Isto vai:
  ✓ Baixar 200 MB de dependências (primeira vez)
  ✓ Compilar o código Java
  ✓ Preparar a aplicação web
  ✓ Gerar JAR executável

⏱️  Tempo: 5-10 minutos (primeira vez)


PASSO 3: EXECUTAR E ACESSAR A PÁGINA WEB
──────────────────────────────────────────

No terminal:

  mvn spring-boot:run

Você verá mensagens como:
  ✓ Starting ECGApplication
  ✓ Tomcat started on port(s): 8080

No navegador, acesse:

  http://localhost:8080

🎉 PRONTO! A página web ECG está rodando!

📡 API REST (Para Integração)
==============================

1. UPLOAD E ANÁLISE

   POST http://localhost:8080/api/ecg/analyze
   
   Parâmetro: file (arquivo CSV)
   
   Resposta:
   {
     "success": true,
     "bpm": 72.5,
     "peakCount": 5,
     "duration": 3.5,
     "minVoltage": -0.4,
     "maxVoltage": 0.18,
     "isAnomalous": false,
     "anomalyDescription": "Ritmo cardíaco normal",
     "voltages": [0.02, 0.05, ...],
     "timestamps": [0, 4, ...],
     "peakIndices": [75, 150, ...]
   }

2. GERAR SIMULADO

   GET http://localhost:8080/api/ecg/simulate?duration=10
   
   Resposta: (mesma do endpoint anterior)

3. HEALTH CHECK

   GET http://localhost:8080/api/ecg/health
   
   Resposta:
   {
     "status": "OK",
     "service": "ECG Analyzer API",
     "version": "1.0.0"
   }

🆘 PROBLEMAS COMUNS
===================

❌ "mvn: comando não encontrado"
   └─ Maven não instalado → Execute: instalar-maven.bat

❌ "Porta 8080 já em uso"
   └─ Use: mvn spring-boot:run -Dspring-boot.run.arguments="--server.port=9090"

❌ "Connection refused"
   └─ Servidor não iniciou → Verifique console para erros

❌ "Arquivo não encontrado"
   └─ Use CSV com formato correto (veja ecg_data.csv)

Mais ajuda → Leia: GUIA_RAPIDO.md
